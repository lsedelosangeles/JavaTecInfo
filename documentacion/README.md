# Programación Orientada a Objetos con Java, para Bachillerato

## Introducción

Este repositorio contiene los archivos, el código, y los documentos para acompañar el curso de [Programación Avanzada][l1], que forma parte del segundo año del **Bachillerato de Tecnologías de la Información**, dictado en [UTU][l2].

Se recomienda encarecidamente combinar esta documentación con los conceptos básicos expuestos en  [**Programación Orientada a Objetos con Java**][l3], ya que esta documentación complementa y expande dicho documento, siendo una continuación del mismo.
